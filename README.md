# Travel Agency Website Overview

## Project Name

**TravelX – Travel Agency Website**

## Project Type

Responsive Travel Agency Landing Page

## Purpose

The website helps users discover travel destinations, view tour packages, book trips, and contact the travel agency.

---

## Features

### 1. Navigation Bar

* Company logo
* Home
* Destinations
* Packages
* Services
* Contact
* Book Now button

---

### 2. Hero Section

Displays:

* Large background image
* Main heading
* Short description
* Explore Tours button

Purpose:

* Attract visitors immediately.
* Encourage users to explore trips.

---

### 3. Popular Destinations Section

Shows destination cards such as:

* Maldives
* Paris
* Venice

Each card contains:

* Destination image
* Destination name
* Short description

---

### 4. Tour Packages Section

Displays available packages.

Examples:

* Beach Escape
* European Tour
* Adventure Trip

Each package contains:

* Package name
* Duration
* Price
* Book button

---

### 5. Services Section

Agency services:

* ✈ Flight Booking
* 🏨 Hotel Reservation
* 🗺 Guided Tours

---

### 6. Testimonials Section

Displays customer reviews.

Example:

> "Amazing experience! Everything was perfectly planned."

Purpose:

* Build trust with visitors.

---

### 7. Contact Section

Includes a form:

* Name
* Email
* Message
* Send button

Purpose:

* Allow users to contact the agency.

---

### 8. Footer

Contains:

```text
© 2026 TravelX. All Rights Reserved.
```

# Responsive Design

The website works on:

* 💻 Desktop
* 📱 Mobile
* 📟 Tablet

Responsive features:

* Navigation stacks on mobile.
* Cards resize automatically.
* Text adjusts to screen size.
